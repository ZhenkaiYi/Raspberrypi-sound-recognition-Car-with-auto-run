# 02 树莓派语音避障小车

#### Description
本项目硬件平台使用树莓派4b，使用Python3开发。
电机驱动使用L289N，语音控制使用USB麦克风，自动避障采用超声波加舵机方案。
调用百度语音识别实现语音控制。
